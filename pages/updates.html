Keep Devices Updated

Keeping devices updated protects against new threats. Turn on automatic updates wherever possible.

  1. Windows: Settings → Windows Update → Turn on automatic updates.
  2. macOS: System Settings → General → Software Update → Automatic updates.
  3. iPhone/iPad: Settings → General → Software Update → Automatic Updates.
  4. Android: Settings → Security & privacy → Updates (varies by phone).
  5. Router: Log into your router’s app or web page → Check for firmware updates. If offered, turn on auto‑updates.
